# Запуск

Перейти в гілку **test**

```
git checkout test
```

# Запуск фронтенда

В терміналі перейти в папку з фронтендом 

```
cd client
```

## Поставити всі залежності 

```
npm i
```
or 

```
yarn
```

## Запустити збірку 

```
npm start
```

or

```
yarn start
```
